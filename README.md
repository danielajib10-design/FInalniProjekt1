Noc v muzeu starých časů
Noc v muzeu starých časů je textová adventura odehrávající se v historické budově Muzea Starých časů během bouřlivé noci.
Hráč se ujímá role Adama, nového nočního hlídače, který má svou první samostatnou směnu.
Krátce po půlnoci je ukradena Svatováclavská koruna a budova se automaticky uzavře nouzovým protokolem.
Adam musí odhalit pachatele a získat korunu zpět, jinak bude označen za hlavního podezřelého.

Cíl hry
Hráč musí postupně plnit úkoly, získávat důvěru ostatních postav, sbírat důležité předměty a vyřešit hádanky, aby se dostal do tajného depozitáře.
Zde se střetne se zlodějem Hubertem. Pokud hráč zloděje porazí a vrátí korunu zpět do vitríny, vyhrává.

Ovládání hry
Hra se ovládá pomocí textových příkazů:

Pohyb
jdi <směr>: přesun do jiné místnosti
Interakce
prozkoumej – detailní popis aktuální lokace
vem <předmět>: sebere předmět
odloz <předmět>: odloží předmět do místnosti a odebere ho z inventáře
pouzij <předmět>: použije předmět z inventáře
oprav <předmět>: opraví kávovar
Komunikace
mluv <postava>: promluví s postavou
Systém
pomoc: zobrazí nápovědu
objev: prozkoumá místnost
napoveda: vypíše nápovědu
inventar: zobrazí inventář
konec: ukončí hru

Herní mechaniky
hráč má inventář s omezenou kapacitou (max. 2 předměty),
postup hrou závisí na plnění úkolů pro ostatní postavy,
některé oblasti jsou přístupné pouze s určitými předměty (např. baterka, klíč),
hra obsahuje závěrečný souboj se zlodějem.

Spuštění hry
Repozitář je možné stáhnout z GitHubu a projekt následně otevřít ve vývojovém prostředí,
kde jej spustíte jako klasickou konzolovou Java aplikaci (spuštěním hlavní třídy programu).

Použité technologie
